 ## 二、介绍
### 1、功能
营救行动是一款LowPoly风格的FPS游戏。
<br>游戏第一关为营救模式，玩家需击败敌人救出人质。游戏第二关为保护模式，添加了塔防元素，玩家需要建造防御塔击败敌人。
